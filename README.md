# AIML Project: Fungal Genetic Risk Analysis

## 📌 Title
Challenges in Regulating Genetic Information in Fungi

## 📖 Description
This project focuses on analyzing the challenges in regulating genetic information in fungi using Artificial Intelligence and Machine Learning techniques.

## 🎯 Objective
To classify fungi as safe or harmful and identify risk patterns using ML models.

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 🤖 Models Used
- Decision Tree Classification
- K-Means Clustering

## 📊 Features Used
- gene_toxin_level  
- growth_rate  
- resistance_index  

## 📈 Outcome
The project helps in predicting fungal risk levels and supports safe regulatory decisions using AI/ML.
